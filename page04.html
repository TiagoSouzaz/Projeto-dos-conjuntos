Diagrama

O diagrama dos conjuntos numéricos é uma representação visual dos diferentes tipos de números que existem. Existem quatro principais conjuntos numéricos: os números naturais, os números inteiros, os números racionais e os números reais.

Números naturais

Os números naturais são aqueles que usamos para contar objetos: 1, 2, 3, 4, 5 e assim por diante. Esses números são representados no diagrama como uma linha reta começando em 1 e continuando infinitamente para a direita.

Numeros inteiros

Os números inteiros são os números naturais junto com seus opostos: -1, -2, -3 e assim por diante. Esses números são representados no diagrama como uma linha reta que inclui os números naturais e se estende infinitamente para a esquerda.

Numeros racionais

Os números racionais são aqueles que podem ser representados como uma fração, onde o numerador e o denominador são números inteiros. Por exemplo, 1/2, 3/4, -2/3 são todos números racionais. Esses números são representados no diagrama como uma linha reta com um ponto marcando cada número racional.

numeros reais

Os números reais são todos os números que podem ser representados na reta numérica, incluindo os números racionais e os números irracionais, como pi e a raiz quadrada de 2. Esses números são representados no diagrama como uma linha reta contendo todos os números racionais e os números irracionais.
